# MkDocs
## Install
```
$ pip install mkdocs
```
## Create project
```
$ mkdocs new <project name>
```

## Run on local
```
$ mkdocs serve
```
- Access to `http://127.0.0.1:8000`

## Build
```
$ mkdocs build
```
