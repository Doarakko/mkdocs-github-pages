# [mkdocs-github-pages](https://doarakko.github.io/mkdocs-github-pages/)
## Requirements
- Python 3
- pip

## Usage
### 1. Download code
```
$ git clone https://github.com/Doarakko/mkdocs-github-pages
$ cd mkdocs-github-pages
```
### 2. Install Python library
```
$ pip install -r requirements.txt
```
### 3. Build
```
$ mkdocs build
```
### 4.  Run on local
```
$ mkdocs serve
```
